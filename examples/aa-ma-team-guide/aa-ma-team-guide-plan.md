# AA-MA Team Guide - Plan

**Objective:** Create comprehensive AA-MA workflow documentation for mixed-experience team, covering both Claude Code and Cursor.

**Owner:** Team
**Created:** 2025-11-26
**Last Updated:** 2025-11-26

## Executive Summary

Two deliverables:
1. Quick Reference Card (2 pages) - workflow diagram + commands cheat sheet
2. Detailed Guide (~30 pages) - progressive depth from beginner to advanced

Structure: Principles first, then tool-specific sections (Claude Code, Cursor).

## Target Audience

- Beginners: Need Quick Start + Quick Reference
- Intermediate: Full workflow (Parts 0-3)
- Advanced: Decision framework + team collaboration (Parts 4-5)

## Pain Points Addressed

1. Context loss between sessions
2. Inconsistent approaches across team
3. Unclear criteria for AA-MA vs ad-hoc

## Key Insight

AA-MA (Claude Code) and Memory Banks (Cursor) are structurally equivalent:
- Same 5-file philosophy
- Same Plan → Execute → Track workflow
- Direct file-to-file mapping exists

## Milestones

1. Foundation (Parts 0-1) - Principles + Quick Start
2. Claude Code (Part 2) - Implementation details
3. Cursor (Part 3) - Implementation details
4. Decision + Team (Parts 4-5) - Framework + collaboration
5. Quick Ref + Appendix - Cheat sheet + FAQ

## Estimated Effort

11-16 hours total across 5 milestones

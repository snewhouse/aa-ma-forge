# Code of Conduct

## The short version

Be kind. Be respectful. Assume good intent.

This is a small project maintained by one person. Everyone who participates, whether by opening issues, submitting pull requests, or commenting, is expected to treat others with basic human decency.

## What that means

- Be patient with response times
- Be constructive with feedback
- Be welcoming to newcomers
- Don't be a jerk

## Enforcement

If someone's behaviour makes this project unwelcoming, I'll address it directly. Persistent bad actors will be blocked.

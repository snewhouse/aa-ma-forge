# DEEPDIVE-TEMPLATES — skeletons for `.claude/onboarding/NN-*.md`

Standard & Deep tiers write these. Each is the *full-detail* version of a section that `ONBOARDING.md`
only summarises. Every file starts with the standard header. Every claim cites evidence. No secrets.

**Standard header (top of every `.claude/onboarding/NN-*.md`):**
```markdown
> Part of the onboarding pack for <repo name> · <YYYY-MM-DDTHH:MM:SSZ> · commit `<short-SHA>` (`<branch>`) · tier: <Standard|Deep> · generated by `understand-codebase`
> Up: [00-index.md](00-index.md) · Root: [../../ONBOARDING.md](../../ONBOARDING.md)
```

The file set:

---
